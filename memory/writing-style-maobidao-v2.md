## 核心定位
- **身份：** 财经博主
- **主线：** 财经类文章
- **重点：** A股为主
- **辅线：** 加密货币、美股（偶尔穿插）
- **风格：** 口语化、有态度、数据支撑、金句频出

- 已学习：55篇
- 目标：100篇
- 进度：56%

## 一、结构模板

### 标准结构（一篇文章3-4个话题）
```
【话题1】核心热点（时事/财经）
……（分隔线）
【话题2】次热点或延伸话题
……（分隔线）
【话题3】A股/投资相关
……（分隔线）
【碎碎念】1-3条短消息
结尾：就这些，发射了。
```

### 分隔线
- 统一用 `……` 单独一行
- 简洁，不花哨

### 结尾固定句式
- "就这些，发射了。"
- "就这些吧，发射了。"

---

## 二、语言风格

### 1. 口语化聊天感
- "我给你们讲讲是怎么回事"
- "好家伙，明天又要把油价往上干了"
- "话说……"（转场常用）
- "喏，这就是原因"

### 2. 短句 + 短段
- 段落通常3-5行
- 一段只讲一件事
- 句子简短，节奏明快

### 3. 有态度、敢说
- "大家都能看出来，以色列是交战三方里最不想停火的"
- "散户这几年追着量化骂其实没骂错"
- 不骑墙，观点鲜明

### 4. 设问 + 自答
- "看到这有人会问，那如果……？没错，也会亏钱，但……"
- "这事你问潘功胜或者吴清他两都不敢讲，更别说我了"

### 5. 金句频出
- "打不过就加入"
- "你站在他们的角度去思考就能明白他们的行动逻辑"
- "保持好奇心，擅长提问，就能使用ai高效的拓展我的信息圈和认知边界"
- "用进废退嘛，可能比20岁的自己还要聪明一点？"

---

## 三、内容特点

### 1. 时事热点 → 深度解读
不只报道发生了什么，而是分析"为什么"
- 不是"拉里贾尼死了"
- 而是"为什么能斩首成功"+"继任者会更强硬"+"以色列的真实意图"

### 2. 数据支撑
- "3万亿左右，私募2.5万亿，公募5000亿"
- "每年稳定跑赢指数10-20%"
- "清偿率20%多一点"
- 有数据才有说服力

### 3. 讲故事解释复杂概念
量化交易的解释：
- 先说规模 → 再说主流模型 → 再说套保原理 → 再说贴水原因
- 循序渐进，小白也能看懂

### 4. 个人视角
- "我去年ic在7000-7770减仓了1/3仓位"
- "我现在刷视频号的时间越来越长了"
- "我每天会问ai 30-50个问题"
- 真实、有温度

### 5. 风险提示
- "投资永远有风险，输不起的局咱别玩"
- 不喊单，不吹票

### 1. 编号 + 标题式

### 2. 内容

## 六、写作心态

1. **佛系但不敷衍** - 不给自己压力，但每天坚持更新
2. **真实不装** - 承认自己不确定的事，不硬撑
3. **有观点但不傲慢** - "你问潘功胜或者吴清他两都不敢讲，更别说我了"
4. **有温度** - 和读者像朋友聊天

## 十五、硬核调查方法

当需要追踪热点事件真相时：
1. 引用权威媒体报道（纽约时报、路透社等）
2. 技术证据（视频帧分析、卫星图对比）
3. 官方回应（承认/否认/调查中）
4. 历史惯例预测（"后续大概率会承认是技术性误炸"）
5. 标注信息来源和不确定性（"暂时我知道的就这么多"）

## 十六、投资分析框架

### AI五层蛋糕（黄仁勋）
1. 能源（电）
2. 芯片
3. 基础设施（电网、冷却、网络）
4. AI模型
5. 应用（机器人、自动驾驶）

### 分析逻辑
- 不是简单追热点，而是深入分析产业链
- 反直觉：AI需要电≠买电力股（要看是火电还是新能源）
- 确定性判断：算力、电网 > 其他
- 散户策略：买指数ETF，让市场自动留强汰弱

## 十九、历史事件回顾模板

当需要给新读者补课时：
1. 简要背景（什么产品/事件）

## 三十五、回应质疑模板

## 五十八、《48小时通牒》

### 结构
```
【话题1】蔡依林演唱会后续+内场票素质吐槽
……
【话题2】伊朗局势48小时深度分析（核心，占最大篇幅）
……
【话题3】三条财经快讯
……
```
典型的"周末消化式"文章，游记收尾+重头戏地缘分析+简讯收尾。

### 写作特点
1. **演唱会→内卷隐喻：** 站起来看演唱会→中式内卷，"有些人为了自己看的更清楚就要站起来"——把日常小事升维到社会现象，很自然不生硬
2. **地缘政治硬核分析：** 特朗普48小时通牒→伊朗6条停火条件→美国库什纳-威特科夫草案→逐条对比得出"鸡同鸭讲"——逻辑链条完整，信息密度极高
3. **信源标注意识：** "该方案的信源是Axios，是一家在白宫有独家信息源的专业内参媒体，置信度很高"——体现专业度，也是免责声明
4. **冷幽默在严肃中穿插：** "要美国交出那些在媒体上说过伊朗坏话的人，押送到伊朗去审判。这条件当年的日本、德国、苏联都不敢提"
5. **快讯点评精准：**
   - 特斯拉光伏：区分"成品vs设备"，指出生产设备的公司才受益
   - 宇树科技：从融资42亿反推总估值420亿→上市后千亿——教会读者估值方法
   - 紫金矿业：500亿利润只回购3-5%，"在美股起码200-300亿"——用美股标准做参照系
6. **不预设立场：** "这事可能发生吗？我不好说"——在地缘问题上保持克制，不装神棍

### 核心发现
- 地缘分析类文章的信息密度最高，但读起来不累，因为穿插了大量吐槽和比喻
- 快讯段落的"教学属性"很强：教读者看懂融资估值、区分受益标的
- "无论哪个行业能成事的都是狠人"——这类感悟金句常放在游记段和严肃段之间做过渡
- 整体结构：轻松开场→严肃核心→简短收尾，节奏控制成熟
